Day3 Advisor & Chat Memory 테스트 페이지

Spring AI의 강력한 Advisor(어드바이저)와 Chat Memory(대화 기억)를 시각적으로 테스트할 수 있는 워크스페이스입니다.

  • Persistent DB Chat: H2 파일 DB를 사용하여 서버를 재시작해도 이전 대화를 기억합니다.
  • In-Memory Chat: RAM에 저장하여 서버가 실행 중인 동안에만 기억을 보존합니다.
  • Simple Chat: 상태를 저장하지 않는 일반 API 호출입니다.
